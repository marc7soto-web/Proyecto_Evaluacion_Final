==========================================
Proyecto Flask con menú y dos ejercicios
=========================================
Proyecto Evaluación Final - Aplicación Flask

Este proyecto contiene una aplicación web desarrollada en Flask que incluye:

- Menú principal con dos ejercicios
- Formulario con ejercicio 1:  Cálculo de compras
- Formulario con ejercicio 2: Autenticación
- Formulario con resultados 1: Muestra los resultados del cálculo de compra y descuentos
- Formulario con resultados 2: Muestra mensaje de bienvenida o error de credenciales
- Estilo CSS personalizado

REQUISITOS PREVIOS
- Python 3.10 o superior
- Flash (La versión indicada en requirements.txt)

INSTALACIÓN
1. Clonar el repositorio desde GitHub:
   git clone https://github.com/marc7soto-web/Proyecto_Evaluacion_Final.git
2. Entrar a la carpeta del proyecto:
   cd Proyecto_Evaluacion_Final
3. Instalar dependencias:
   pip install -r requirements.txt

EJECUCIÓN
1. Ejecutar la aplicación:
   python main.py
2. Abrir el navegador en:
   http://127.0.0.1:5000
